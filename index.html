Installing burndown

Since burndown is not in the official R repository, you'll need to install it yourself. Do that by typing the following in an R session.

library(devtools)
install_github("burndown", "dankelley", "master")

You'll need to first do

install.packages("devtools")

if you don't have that installed already.
